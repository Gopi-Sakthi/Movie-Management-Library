Movie Management Library

  * A comprehensive Java library for managing movies and related data. It simplifies the handling, categorization, and retrieval of movie information for applications, services, or systems that require robust movie management features.

Tools & Technology used

- Java
- Servlet [server side].
- JSP Pages [client side].
- JDBC for database connectivity.
- Apache Tomcat server
- MySQL for data storage.

Features

* Add, update, and remove movies
* Search and filter by title, genre, actor, year, and more
* Store and manage metadata such as ratings, reviews, and posters
* Basic user authentication and management

Getting Started

- Prerequisites

- Java 8 or higher
- [Maven](https://maven.apache.org/) or [Gradle](https://gradle.org/) (for dependency management)

Installation

You can include this library in your Java project by cloning the repository and building it yourself:

```bash
git clone https://github.com/Gopi-Sakthi/Movie-Management-Library.git
cd Movie-Management-Library
mvn clean install
```
